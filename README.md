# AyikSolution — AI Automation & Web Agency

AyikSolution is an agency portfolio and product website built with React, TypeScript, Vite and Tailwind-style utility CSS. The site showcases services (AI booking automation, social media, ads, reputation management and analytics) and includes a lead capture form that posts to Formspree.

**What this repo contains**
- **Frontend:** a modern, responsive single-page React app located in the project root `src/` folder.
- **Design & UX:** animated hero, service cards, pricing, team/work sections, contact form and modal interactions.

## Key Features
- AI-focused messaging and positioning targeted at clinics and small businesses.
- Responsive, animated UI with reveal-on-scroll and parallax effects in the hero.
- Services carousel and detailed service cards with images and CTAs.
- Pricing section with monthly retainer examples and clear CTAs.
- Contact form with client-side validation that submits to Formspree (see `src/components/ContactStrip.tsx`).
- Modular React components for easy customization: `Navbar`, `Hero`, `Services`, `Pricing`, `Work`, `Reviews`, `Teams`, and `Footer`.

## Tech Stack
- React + TypeScript
- Vite (development server & build)
- Tailwind-like utility classes + custom CSS (see `src/index.css`)
- Framer Motion (animations)
- Formspree for contact form submissions

## Quick Start
Requirements: Node.js (16+ recommended) and npm or yarn.

Install dependencies:

```
npm install
```

Run dev server:

```
npm run dev
```

Build for production:

```
npm run build
```

Preview production build locally:

```
npm run dev
```

## Important Implementation Notes
- The contact form posts to a hard-coded Formspree endpoint: `https://formspree.io/f/*******` (see `ContactStrip.tsx`). Replace this with your own endpoint or backend route for production.
- Services and cards are defined in `Services.tsx` as an array of objects — update that array to change images, copy, or features.
- Intersection Observer is used in `App.tsx` to add the `visible` class to elements with `.reveal` for on-scroll animations.

## Customization Checklist
- Replace placeholder Unsplash images in `Services.tsx` with your brand assets.
- Update copy in `Hero.tsx`, `Services.tsx`, and `Pricing.tsx` to reflect current offerings and prices.
- Swap the favicon and open-graph image in `index.html` (`/favicon.jpeg`, `/og-image.png`).
- If you need server-side form handling, replace the Formspree POST in `ContactStrip.tsx` with your API endpoint.
